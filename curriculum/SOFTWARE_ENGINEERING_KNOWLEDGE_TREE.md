# Software Engineering Knowledge Tree

Status: Draft

### Purpose:

Define the recommended learning sequence for becoming a Software Engineer, preparing for the MInfoTech programme, and building foundations for future AI Engineering studies.

# Learning Phases

## Phase 0 - Foundations

### Purpose:

Build the computer literacy, development environment, and problem-solving skills required for all future phases.

### Topics:

- Computer Hardware Basics
- Operating Systems
- Linux Fundamentals
- Command Line
- Files and Directories
- Internet Fundamentals
- Networking Fundamentals
- Development Environments
- Problem Solving

### Competencies:

- Navigate a Linux system confidently
- Use the command line effectively
- Understand how computers, operating systems, and networks work
- Set up a development environment
- Troubleshoot common technical issues
- Use GitHub and basic Git workflows

### Evidence of Competence:

- Linux environment configured
- GitHub profile configured
- SSH keys configured
- Git workflow demonstrated
- Command line exercises completed
- Personal development environment documented

### Suggested Projects:

- Linux Setup Guide
- Command Line Cheat Sheet
- Development Environment Documentation
- Networking Notes

### Completition Criteria:

- Can comfortably use Linux and the terminal

- Can clone, commit, push, and pull using Git

- Can configure a development environment independently

- Can explain basic networking concepts

- Can troubleshoot common system issues
